# 📈 AI Stock Analyzer

A simple AI-powered stock analysis web app built using Streamlit.

## 🚀 Live App
👉 https://ai-stock-analyzer-9fjrybzv5bp6kg3pn5trwu.streamlit.app/

## 🔍 Features
- RSI Indicator
- Moving Average (MA20)
- Support & Resistance
- Buy / Sell Signals
- Candlestick Analysis
- Volume Analysis

## 🛠️ Tech Stack
- Python
- Streamlit
- yFinance
- Pandas
- Matplotlib

## 📊 How to Use
1. Enter NSE stock symbol (e.g. INFY.NS)
2. Click "Analyze Stock"
3. View signals and charts
